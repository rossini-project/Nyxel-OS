COMO INSTALAR

Nyxel OS — Edição Claude Code (a avançada)

Esta é a edição avançada, que roda dentro do Claude Code com memória em arquivos e automações. Se você quer o caminho mais fácil e sem instalar nada, use a Edição Universal (pasta Edicao-Universal) — ela funciona em qualquer IA.

Este guia instala a edição Claude Code do zero — sem nada instalado antes. Não precisa saber programar. É só seguir os passos na ordem. Leva uns 15 minutos.

Antes de começar — o único requisito

Pra usar o Claude (o cérebro do sistema), você precisa de uma assinatura paga do Claude: o plano Pro ou Max, em claude.com. Sem isso, o sistema não funciona. Não é preciso chave de API nem nada técnico — é só a assinatura normal.

PASSO 1

Instalar o VS Code

O VS Code é um programa gratuito da Microsoft — é a 'casa' onde o Claude e o Nyxel OS vão morar.

Abra seu navegador e vá em code.visualstudio.com. Clique no botão azul/verde de baixar (ele já detecta se você usa Windows ou Mac).

Quando baixar, abra o arquivo e instale como qualquer programa: no Windows, vá clicando em 'Avançar' até 'Concluir'; no Mac, arraste o ícone do VS Code para a pasta Aplicativos.

PASSO 2

Abrir o VS Code e achar as Extensões

Abra o VS Code. Na faixa de ícones do lado esquerdo, procure o ícone de 'quadradinhos' (quatro quadradinhos) — é o de Extensões.

Pode também usar o atalho do teclado: Ctrl+Shift+X no Windows, ou Cmd+Shift+X no Mac.

PASSO 3

Instalar a extensão Claude Code

No campo de busca das Extensões, digite: Claude Code.

Vai aparecer a extensão oficial, da Anthropic. Confira o nome do autor (Anthropic) e clique em Instalar. Espere uns segundos até terminar.

Se aparecer mais de uma, escolha a da Anthropic — é a oficial.
PASSO 4

Entrar na sua conta Claude

Depois de instalar, clique no ícone do Claude que apareceu na faixa lateral. Na primeira vez, ele abre o navegador pra você entrar na sua conta Claude.

Faça login com a conta da sua assinatura (Pro ou Max). Pronto, o Claude está conectado ao VS Code.

Lembrete: é a assinatura paga do Claude. Não precisa de chave de API.
PASSO 5

Abrir a pasta da Edição Claude Code

Descompacte o arquivo .zip do Nyxel OS que você baixou (clique com o botão direito → Extrair).

No VS Code, vá no menu Arquivo → Abrir Pasta… e escolha a pasta Edicao-Claude-Code (ela está dentro da pasta Nyxel OS). É essa pasta que contém o sistema.

Abra a pasta Edicao-Claude-Code — é ela que tem o cérebro do sistema.
PASSO 6

Ligar o Nyxel OS

Com a pasta Edicao-Claude-Code aberta, abra a conversa do Claude (o ícone na lateral), digite /instalar e aperte Enter.

O sistema vai te entrevistar, uma pergunta por vez, sobre o seu negócio. Responda com calma. Quando terminar, o Nyxel OS está configurado com a sua cara — roda uma vez só.

Depois do setup, é só usar. Sempre que abrir a pasta no VS Code, o painel do dia aparece.
PASSO 7

Usar no dia a dia

Sempre que for trabalhar, abra a pasta do Nyxel OS no VS Code — o sistema já te mostra o painel do dia.

Comandos pra começar: /painel (seu dia inteiro), /proposta (proposta pra cliente), /cobranca (cobranças no tom certo), /salvar (backup do seu trabalho).

A lista completa das 27 skills está no arquivo CATALOGO.md.

SE TRAVAR

Problemas comuns

O ícone do Claude não apareceuFeche e abra o VS Code de novo. Confira em Extensões que o Claude Code está instalado.
Pede pra fazer login de novoNormal na 1ª vez. Faça login com sua conta paga do Claude no navegador que abrir.
Não sei qual pasta abrirAbra a pasta Edicao-Claude-Code (dentro de Nyxel OS). É a que tem os arquivos do sistema.
Digitei /instalar e não aconteceu nadaConfira que você está na conversa do Claude (não no editor de texto) e que a pasta Edicao-Claude-Code está aberta.
