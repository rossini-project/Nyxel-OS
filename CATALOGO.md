# Catálogo de Skills — Nyxel OS

Referência rápida das 27 skills. Digite `/` + o nome em qualquer sessão.

## Núcleo — operação do dia a dia
| Skill | O que faz |
|---|---|
| `/instalar` | Setup inicial: entrevista o negócio e configura tudo (roda 1x) |
| `/abrir` | Carrega o contexto e dá um resumo curto pra começar |
| `/painel` | Painel executivo do dia: foco, clientes, dinheiro, metas, alertas |
| `/foco` | Aponta a próxima ação de maior impacto agora |
| `/decidir` | Decide com base na estratégia, caixa e metas reais |
| `/salvar` | Commit + push no GitHub (backup do trabalho) |
| `/atualizar` | Varre o workspace e reconcilia a memória |
| `/novo-projeto` | Cria pasta de cliente/projeto e registra no CRM |
| `/mapear-rotinas` | Transforma tarefas repetidas em skills suas |
